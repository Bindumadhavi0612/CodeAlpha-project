# Disease Prediction using Machine Learning

## Overview

This project develops a machine learning model to predict diseases based on patient symptoms and medical data. Various classification algorithms were implemented and evaluated to determine the most accurate prediction model.

## Objectives

* Preprocess and analyze healthcare datasets.
* Train multiple machine learning models.
* Compare model performance using evaluation metrics.
* Predict diseases accurately from input symptoms.

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Matplotlib
* Seaborn

## Machine Learning Algorithms

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* XGBoost

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

## Results

Among the implemented models, XGBoost achieved the highest accuracy of approximately 86%, demonstrating strong performance for disease prediction.
